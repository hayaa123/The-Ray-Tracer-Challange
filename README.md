# The Ray Tracer Challange

This repository contains my implementation of The Ray Tracer Challenge book.

You may notice that this repo includes my own libft library. This is because I am practicing for the 42 MiniRT project, but you can also use ready-made functions.

## Chapter 1
- Tuple, vector and point.
- Tuple mathematics: addition, subtraction, negation, scaling, and division.
- Vector operations: magnitude, normalization (converting to a unit vector), dot product, and cross product.

## Chapter 3
- Matrix creation.
- Matrix operations (multiplication, comparison).
- Identity matrix and transpose operation.
- Matrix inversion (including the calculation of determinants, cofactors, and minors, where a minor is the determinant of a submatrix at a given position).

# How to run the code?

- First, clone the repository:

```git clone git@github.com:hayaa123/The-Ray-Tracer-Challange.git```

- Move into the cloned directory:

```cd The-Ray-Tracer-Challange```

- Compile the project:
```make```

- Run the executable:
``` ./minirt ```
