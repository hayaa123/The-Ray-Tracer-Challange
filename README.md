# The Ray Tracer Challange

This repository is my own code following the The Ray Tracer Challange book.
You can notice that this repo has my own library called libft this is because I am practicing 
it for 42 minirt project but you can use ready functions.

# Chapter 1
- Tuple, vector and point.
- Tuple mathmatics: addition, substraction, negation, scaling and division.
- Vector operations: magnitude, normalization (convert to unit vector), dot product and cross product.

# How to run the code?

- First clone the repo.

```git clone git@github.com:hayaa123/The-Ray-Tracer-Challange.git```

- Move into the cloned directory.

```cd The-Ray-Tracer-Challange```

- Compile the project by running 
```make```

- Run the executable file ./minirt
